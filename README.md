# Homebrew Tap

Personal Homebrew tap for my tools.

## Installation

```bash
brew tap manashmandal/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| imggen  | CLI tool for generating images using AI image generation APIs |

## Usage

```bash
# Install imggen
brew install manashmandal/tap/imggen

# Or after tapping
brew tap manashmandal/tap
brew install imggen
```
